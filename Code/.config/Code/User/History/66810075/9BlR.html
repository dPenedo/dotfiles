Iniciar un proceso de mediación

¿En qué consiste un proceso de mediación?

La mediación es un intento de trabajar con el otro y no contra el otro, en busca de una vía pacífica y equitativa para afrontar los conflictos, en un entorno de crecimiento, de aceptación, de aprendizaje y de respeto mutuo.

Consiste en un proceso voluntario, flexible y participativo de resolución pacífica de conflictos, en el que dos partes enfrentadas recurren voluntariamente a una tercera persona imparcial, el mediador, para llegar a un acuerdo satisfactorio. Este método promueve la búsqueda de soluciones que satisfagan las necesidades de las partes.

¿Cómo puedo iniciar un proceso de mediación?

Contacta directamente por correo-electrónico o por whatsapp para consultar por el primer encuentro. La mediación se puede realizar tanto de manera on-line como de manera presencial en Barcelona y en el Baix Llobregat.
